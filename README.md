# Real-Time Chat Application

## 📌 Description
This is a real-time chat application built using Spring Boot and WebSocket.

## 🚀 Features
- Real-time messaging
- Instant updates without page refresh
- Multiple users can chat

## 🛠 Technologies Used
- Java (Spring Boot)
- WebSocket (STOMP, SockJS)
- HTML, CSS, JavaScript
- Bootstrap

## ▶️ How to Run
1. Start the Spring Boot application
2. Open the HTML file in browser
3. Enter your name and message
4. Click Send

## 📷 Output
Chat messages appear instantly in the chat box.
